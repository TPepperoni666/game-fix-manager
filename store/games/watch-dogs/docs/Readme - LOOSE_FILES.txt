!!! ONLY FOR MODDERS !!!

This special variation of the Living_City mod is based on the latest version and is specifically 
made to help users in creating their own merges of this mod with other graphic mods.
This means that you need at least some basic knowledge about Watch_Dogs modding if you wanna make any 
use of this package, because this is not the usual ready-to-use version of the mod.
This package contains only the files related to my mod, without any other vanilla patch file, so you 
must ensure that the other mod already includes the vanilla patch or you'll have to add it yourself.



The content is split in three main folders:


- REQUIRED_MOD_FILES: contains the CORE mod files and all the ones that are NOT related to graphics.
(no Beretta 92 or Dynamic Headlight Shadows or GasStation Debris or E3 2012 Focus or Relief Mapping etc)

- OPTIONAL_GRAPHIC_TWEAKS: contains NO files of the main mod except the ones related to the graphic tweaks.
(so this folder does NOT contain my mod, but only the little graphic tweaks included with it)

- OPTIONAL_RELIEF_MAPPING: contains ONLY the files for the Relief Mapping graphic feature and nothing else.
(so this folder does NOT contain my mod, NOR any other graphic tweaks except this specific feature)


Basically these three folders contains the full Living_City mod split in pieces ready to be merged with 
whatever mod you want, but be aware that this doesn't give ANY guarantee about compatibility and there's 
still a significant chance that you'll see bugs, especially with really outdated mods like TheWorse.
